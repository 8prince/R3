# R3
## Rakuten Real time Rescue

### For
- Tourists who want help
- Locals who want to help tourists

### Features
- Real time chat
- Real time notification
- Navigation

### Usage
- Docker
    - `./docker-run.sh`
- Maven
    - `JDK8+` & `Maven`
    - `mvn clean install`
    - `java -jar target/internship-0.0.1-SNAPSHOT.jar`
